# VkNewsFeed

Simple alternative Vk client
![](https://github.com/TopIvanAbramov/VkNewsFeed/blob/master/VKNews/AppPhotos/appDemo.png)
### Features

-Sign in with VK\
-Read posts with: text, photos or link attachments\
-Search new posts

### Architecture

[CleanSwift](https://clean-swift.com)

## Used frameworks

-VK-ios-sdk\
-SkeletonView
